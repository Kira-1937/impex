# Impex Feature Roadmap

## Phase 1: MVP (Minimum Viable Product)
*Goal: Establish the core marketplace and basic group buying functionality.*

*   **Auth:** Google Login for Buyers and Sellers.
*   **Marketplace:** Basic product listing (Sellers) and browsing/search (Buyers).
*   **AI Chat:** Basic Gemini-powered assistant for general product queries.
*   **Simple Groups:** Manual creation of purchase groups by country.
*   **Basic Orders:** Direct purchase flow without complex logistics integration.

## Phase 2: V2 (Optimization & Trust)
*Goal: Improve the collaborative experience and build platform trust.*

*   **Seller Verification:** Badge system for verified Indian exporters.
*   **Advanced Grouping:** Automated group suggestions based on buyer location.
*   **Cost Calculator:** Real-time estimation of shipping savings when joining a group.
*   **AI Export Pro:** Specialized AI guidance for documentation (Invoices, Packing Lists).
*   **Order Tracking:** Milestone-based tracking (Warehouse -> Customs -> International -> Local).

## Phase 3: V3 (Scale & Automation)
*Goal: Global scale and seamless logistics.*

*   **Logistics Integration:** API connections with global carriers (DHL, FedEx) for real-time rates.
*   **Multi-currency Support:** Dynamic pricing based on buyer's local currency.
*   **Admin Dashboard:** Advanced tools for dispute resolution and platform analytics.
*   **Mobile App:** Dedicated iOS/Android versions for on-the-go trading.
*   **Community Features:** Buyer reviews and seller ratings to build a self-sustaining ecosystem.
